# Taller de Recursividad

## Nombre del estudiante
Carlos Mario Bucheli Prado

## Descripción del taller
Este proyecto contiene 14 ejercicios de recursividad desarrollados en Java utilizando NetBeans.

El taller está organizado en paquetes según el tipo de problema recursivo:
- `basic_Recursion`
- `recursion_Array`
- `recursion_Matrix`
- `advanced_Recursion`

Los ejercicios incluyen:
- factorial
- sumatoria hasta n
- serie armónica
- invertir número
- suma de dígitos
- potencia
- máximo común divisor (MCD)
- copia de cadenas
- división entera
- multiplicación
- suma de arreglo
- suma de matriz
- Fibonacci
- Ackermann

El objetivo de este taller es aplicar funciones recursivas, casos base y llamadas recursivas usando código limpio y principios de programación orientada a objetos.

## Instrucciones para ejecutar el programa
1. Abrir el proyecto en NetBeans.
2. Abrir el paquete del ejercicio que se desea ejecutar.
3. Ejecutar la clase `View` correspondiente:
   - `ViewBasicRecursion`
   - `ViewRecursionArray`
   - `ViewRecursionMatrix`
   - `ViewAdvancedRecursion`
4. Ingresar los valores solicitados en consola.
5. Revisar el resultado en la terminal de salida.
